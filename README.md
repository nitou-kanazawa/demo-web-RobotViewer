# Robot Viewer

バイブコーディングで作る、Webで動くデモアプリ置き場。スマホでの確認を前提にしています。

## 🤖 多関節ロボット シミュレータ（`index.html`）

three.js + [urdf-loader](https://github.com/gkjohnson/urdf-loaders) で、URDF で定義した多関節ロボットをブラウザに表示し、関節をスライダーで動かせるデモ。

- **単一HTMLファイル + CDN** — ビルド不要。ファイルを開くだけ／GitHub Pages にそのまま置ける
- **スマホ最適化** — 指で回転・ピンチでズーム、画面下のパネルで操作（「関節」「パス」タブ）
- **サンプル6軸アーム同梱** — プリミティブ形状の URDF を内蔵しているので外部メッシュ不要
- **静的パス再生** — 関節角の姿勢を経由点として登録 → なめらかに補間して連続再生／ループ可、手先の軌跡も表示

### 使い方（パス再生）
1. 「関節」タブでスライダーを動かして姿勢を作る
2. 「＋ この姿勢を経由点に追加」で経由点として登録（複数回）
3. 「パス」タブで **▶ 再生** … 経由点を順になぞって動きます
   - 区間あたりの秒数、ループ、軌跡表示を切り替え可能
   - 各経由点は ⤓（その姿勢へ移動）／✕（削除）

### 開発ロードマップ
- [x] **Phase 1** … URDF 表示／関節スライダー手動操作／タッチ操作
- [x] **Phase 2** … 静的パスシミュレータ（関節角の経由点を登録 → 補間再生）
- [ ] **Phase 3** … 環境（障害物・床）との当たり判定

### ローカルで開く
CDN を使うため、ファイルを直接開くのではなく簡易サーバー経由が安全です。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000/ を開く
```

### GitHub Pages で公開（スマホ確認用）
リポジトリの Settings → Pages → Branch を対象ブランチに設定すると、
`https://<user>.github.io/demo-viewer/` で開けます。
